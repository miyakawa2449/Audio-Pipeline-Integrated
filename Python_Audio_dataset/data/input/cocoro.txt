私はその人を、いつも先生と呼んでいました。
だからこの文章でも、本名は明かさずに先生とだけ書くことにします。
誰かに遠慮しているわけではありません。
ただ、その方が自分にとって、とても自然なことなのです。
先生を思い出すたび、私の口からは自然と「先生」という言葉が出てきます。
こうしてペンを握っている今も、その気持ちはまったく同じです。
イニシャルで呼ぶなんて、よそよそしくて考えられません。

先生と私が出会ったのは、鎌倉の海でした。
その頃の私は、まだ血気盛んな学生でした。
夏休みを利用して海水浴に行っていた友人から、「ぜひ来い」とLINEが届いたのです。
私はなんとかお金を準備して、鎌倉へ向かうことにしました。
お金の工面に２、３日かかってしまいました。
ところが、私が鎌倉に着いて３日も経たないうちに、事態は急変します。
私を呼び寄せたその友人あてに、実家から「直ぐに帰れ！」とSMSが届いたのです。
LINEには「母が病気だから」と書かれていましたが、友人はそれを信じていませんでした。
実は彼、親から望まない結婚をずっと強制されていたのです。
今の感覚で言えば、結婚するにはまだ若すぎました。
それに何より、肝心の相手のことが気に入らなかったのです。
だから、夏休みになっても実家に帰らず、わざと東京の近くで遊んでいたというわけでした。
彼はLINEを私に見せて、「どうしよう」と相談してきました。
私に良い考えなど浮かぶはずもありません。
でも、もし本当にお母さんが病気なら、帰るべきです。
結局、友人は実家へ帰ることになりました。
誘われてやってきた私は、たった一人、取り残されてしまったのです。

学校が始まるまでには、まだずいぶん時間がありました。
このまま鎌倉に残ってもいいし、東京に帰ってもいい。
そんな状況の私は、しばらく今の宿に泊まり続けることに決めました。
友人は地方の資産家の息子で、お金には困っていない男でした。
それでも、学生という身分なので、暮らしぶりは私と大して変わりませんでした。
だから一人になっても、わざわざ宿を変えるのは面倒だったのです。

その宿は、鎌倉の中でも少し外れた場所にありました。
ビリヤードやアイスクリームといった流行りの店が並ぶ街の中心部へは、田んぼのあぜ道をいくつも越えなければなりません。
車で行けばすぐですが、かなりのお金がかかりました。
ただ、その代わり、周囲には個人の立派な別荘がいくつも建ち並んでいました。
それに海が目と鼻の先だったので、海水浴には最高のロケーションだったのです。

私は毎日、海へ出かけました。
まるで昔話に出てくるような、古い茅葺き屋根の家の間を抜けて浜辺へ出ると、そこは別世界でした。
「この辺りにこんなに都会の人がいたのか」と驚くほど、避暑にやってきた男女で砂浜が埋め尽くされていたのです。
海の中も、まるで芋を洗うように、たくさんの人の頭でごった返していることもありました。
もちろん、知り合いなんて一人もいません。
それでも、この賑やかな雰囲気に包まれて、砂浜に寝転んだり、波打ち際で水と戯れたりするのは、とても愉快な時間でした。

先生を初めて見つけたのは、まさにこの雑踏の中でした。
その頃、海岸には浜茶屋が２軒ありました。
私は、ひょんなことからそのうちの１軒をよく利用するようになっていました。
長谷あたりに大きな別荘を持っている人たちとは違い、この辺りの海水浴客は自分専用の更衣室なんて持っていません。
だから、こうした共同の着替え場所がどうしても必要だったのです。
みんな、ここでお茶を飲んで休憩します。
それだけでなく、水着を洗ってもらったり、シャワー代わりに体を流したり、帽子や傘を預けたりもしていました。
私は水着を持っていませんでしたが、それでも荷物を盗まれる心配はあります。
だから海に入るときはいつも、その茶屋にすべての荷物を預けるようにしていました。

私がその浜茶屋で先生を見かけたとき。
先生はちょうど服を脱いで、これから海に入ろうとするところでした。
一方の私は、海から上がってきたばかりで、濡れた体に潮風を受けていました。
私たちの間には、視界を遮るようにたくさんの人の頭が動いています。
何か特別なことがなければ、私は先生の存在に気づかずに通り過ぎてしまったでしょう。
あれだけ浜辺が混み合っていたのに。
それに、私自身もただぼんやりしていただけなのに。
それでも先生をすぐに見つけ出せたのは、一人の西洋人を連れていたからでした。
その西洋人の、際立って白い肌の色が、浜茶屋に入ってきた瞬間、パッと私の目を引いたのです。
彼は純和風の浴衣を着ていましたが、それを無造作にベンチの上に脱ぎ捨てました。
そして腕を組んだまま、じっと海の方を向いて立っています。
彼が身につけていたのは、日本でいう「ふんどし」一枚だけでした。
それが、私にはまず不思議でなりませんでした。

実はその２日前、私は由比ヶ浜まで足を伸ばしていました。
砂浜に座り込み、西洋人たちが海水浴をする様子を長いこと眺めていたのです。
私が座っていたのは、ホテルの裏口に近い、少し小高い場所でした。
そのため、たくさんの西洋人の男女が海水浴に出てくるのを目にしました。
しかし男性は皆、胴や腕、太ももまで隠すタイプの水着を着ていました。
女性たちは、さらに肌を隠す傾向がありました。
ほとんどがゴム製の帽子をかぶり、地味な色の水着で波間に浮かんでいたのです。
そんな光景を見たばかりの私にとって、ふんどし一枚で堂々と人前に立つこの西洋人は、とても珍しい存在に見えました。

やがて彼は、自分の隣にしゃがみ込んでいる日本人に、一言、二言、何か話しかけました。
その日本人は、ちょうど砂の上に落ちた手ぬぐいを拾い上げているところでした。
彼はそれを拾うと、すぐにそれで頭を覆い、海の方へ歩き出します。
その人こそが、先生だったのです。

私はただの好奇心から、並んで浜辺を歩いていく二人の後ろ姿を目で追っていました。
二人はまっすぐに波の中へ入っていきます。
そして、岸辺で騒いでいる大勢の人の間を抜け、比較的空いている場所まで行くと、二人そろって泳ぎ始めました。
彼らは、頭が豆粒のように小さく見えるくらいまで、沖へ向かって泳いでいきます。
そして、そこからUターンすると、また一直線に浜辺まで戻ってきました。
浜茶屋に戻ると、二人は井戸水で体を流すこともしません。
すぐに体を拭いて服に着替えると、さっさとどこかへ行ってしまいました。

二人が出て行った後も、私は元のベンチに座ってタバコをふかしていました。
その時、私はぼんやりと先生のことを考えていました。
どうにも見覚えのある顔のような気がしてなりません。
しかし、いつ、どこで会った人なのか、どうしても思い出せずにいました。

当時の私は、悩みがないというより、むしろ退屈を持て余していたのです。
だから翌日も、先生に会ったのと同じくらいの時間に、わざわざ浜茶屋まで出かけてみました。
すると、その日は西洋人の姿はなく、先生が一人で麦わら帽子をかぶってやって来ました。
先生は眼鏡を外して台の上に置くと、手ぬぐいで頭を覆い、さっさと浜辺へ下りていきます。
先生が昨日と同じように、賑やかな海水浴客の間を抜けて一人で泳ぎ始めたとき。
私は急に、その後を追いかけたくなりました。
私は浅瀬を駆け抜け、水しぶきを上げながらある程度の深さまで進みます。
そこから、先生を目指してまっすぐに泳ぎ始めました。
ところが先生は、昨日とは違って弧を描くように泳ぎ、予想外の方向から岸へ戻り始めたのです。
そのせいで、私のささやかな目的は、あっけなく失敗に終わりました。
私が陸に上がり、腕から滴る水を振りながら浜茶屋に入ると、先生はすでに着替えを終えていました。
そして、私と入れ違いに、黙って外へ出て行ってしまったのです。
